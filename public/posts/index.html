What is Hugo

Hugo is a fast and modern static site generator written in Go, and designed to make website creation fun again.

This website is hosted on GitHub Pages, powered by Hugo with Console Theme, and is best viewed using Internet Explorer on a machine running Windows Vista (Service Pack 2) with an Intel(R) processor (no AMD allowed) and at a screen resolution of 1024 x 768 with at least 16,384 colours. Machines not meeting this specification will be shot.
